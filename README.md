# 批量文件重命名工具 (Batch File Renamer)

零依赖的 Python 批量文件重命名工具，带 Web UI，支持预览、撤销、正则表达式。

![Python](https://img.shields.io/badge/Python-3.6+-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 功能

- **多种命名模式**：添加前缀/后缀、查找替换、正则替换、序号命名、更改扩展名、大小写转换、删除字符、插入文本
- **应用前预览**：表格对比显示原文件名与新文件名
- **冲突检测**：自动检测重名冲突并标记
- **一键撤销**：每次操作记录历史，可撤销还原
- **大批量处理**：分页显示，高效处理 1000+ 文件
- **保留元数据**：使用 `os.rename`，不改变文件权限、时间戳等
- **规则叠加**：可添加多条规则按顺序依次生效
- **零依赖**：仅使用 Python 标准库，无需安装任何第三方包

## 快速开始

```bash
python3 batch_renamer.py
```

浏览器会自动打开 `http://127.0.0.1:8765`。

## 使用方法

1. **输入文件夹路径**，点击「加载文件」
2. **添加命名规则**（可多条叠加）
3. **查看预览表格**，确认变化
4. 点击 **「应用重命名」**

## 命名模式说明

| 模式 | 说明 | 示例 |
|------|------|------|
| 添加前缀 | 在文件名前添加文本 | `photo.jpg` → `2024_photo.jpg` |
| 添加后缀 | 在扩展名前添加文本 | `photo.jpg` → `photo_backup.jpg` |
| 查找替换 | 替换文件名中的文本 | `IMG_001.jpg` → `photo_001.jpg` |
| 正则替换 | 使用正则表达式替换 | `file (1).txt` → `file_1.txt` |
| 序号命名 | 按顺序编号 | `abc.jpg` → `photo_001.jpg` |
| 更改扩展名 | 批量改扩展名 | `file.jpeg` → `file.jpg` |
| 大小写转换 | 转换大小写 | `FILE.TXT` → `file.txt` |
| 删除字符 | 删除指定字符 | `file - copy.txt` → `filecopy.txt` |
| 插入文本 | 在指定位置插入 | `photo.jpg` → `my_photo.jpg` |

## 正则表达式示例

```
# 去掉括号和空格：file (1).txt → file1.txt
模式: \s*\(\d+\)
替换: 空

# 提取日期重组：20240101_photo.jpg → 2024-01-01_photo.jpg
模式: (\d{4})(\d{2})(\d{2})_(.+)
替换: $1-$2-$3_$4
```
